# Oil Market Analysis

## 📊 Project Overview
This project analyzes global oil production and geopolitical trends using real-world datasets.

## 📁 Dataset
- Oil country production data
- Geopolitical oil dataset (2010–2026)

## 🛠 Tools Used
- Python
- Pandas
- Jupyter Notebook

## 🔍 Key Analysis
- Data cleaning and preprocessing
- Trend analysis of oil production
- Insights on geopolitical impact

## 📌 Results
- Identified patterns in oil production over time
- Highlighted key countries influencing the market

## 🚀 How to Run
1. Open the notebook
2. Run all cells
3. Make sure datasets are in the same folder
